## 🧠 Overview

LectureSnap ULTRA is a powerful browser-based system that captures lecture screens, intelligently filters important frames, and converts them into clean, structured study material.

Built for students who rely on online learning platforms like YouTube but don’t receive proper notes.

---

## 🎯 Problem

Students watching online lectures face:

* ❌ No structured notes provided
* ❌ Time wasted in manual note-making
* ❌ Messy, unordered screenshots
* ❌ Difficult revision process

---

## 💡 Solution

LectureSnap ULTRA automates the entire workflow:

> Capture → Select → Organize → Export

* 📸 Auto screenshot capture
* 🧠 Smart filtering (dedup + blur detection)
* 🗂 Organized timeline + chapters
* 📄 Export into clean notes (PDF, PPT, Markdown, etc.)

---

## ⚡ Features

### 🎥 Capture System

* Screen capture via browser
* Adjustable interval (3–60 sec)
* Manual snap support
* Region-based capture
* Scheduled capture

---

### 🧠 Smart Processing

* Duplicate frame detection
* Motion blur filtering
* Importance scoring
* Smart skipping of similar frames

---

### 🗂 Organization

* Timeline view
* Chapter tagging & auto-detection
* Pin / Star important frames
* Search & filter system

---

### ✏️ Editing Tools

* Annotation editor (draw, text, highlight, blur)
* Crop & adjust (brightness, contrast, saturation)
* Frame comparison mode
* Notes per screenshot

---

### 🔍 OCR (Text Extraction)

* Extract text from screenshots
* Make notes searchable

---

### 📤 Export Options

* 📦 ZIP (ordered images)
* 📄 PDF (1/2/4 per page)
* 📊 PPTX (presentation ready)
* 🌐 HTML gallery
* 📝 Markdown (Obsidian-ready)
* 📱 QR sharing

---

### ⚙️ Advanced Controls

* Quality adjustment
* Format selection (JPEG / PNG / WebP)
* Watermark support
* Auto PDF generation
* Floating capture counter

---

## 🏗 Tech Stack

### Frontend

* HTML5, CSS3, JavaScript (Vanilla)

### Core APIs

* Screen Capture API (`getDisplayMedia`)
* Canvas API (frame extraction)

### Libraries

* JSZip → ZIP export
* jsPDF → PDF generation

---

## ⚙️ How It Works

1. Start screen capture
2. Tool captures frames at set interval
3. Smart engine filters unnecessary frames
4. Frames appear in grid + timeline
5. Select / edit important frames
6. Export as structured notes
